# Lab 5: EDA and Data Visualization

## Repo Purpose
This is a lab created for the BU RISE Data Science Practicum as data analysis practice. This repo takes a small sample of students, and plots the correlation between the students' ages and their GPAs. 

## Environment Setup
pytest
pandas
matplotlib
seaborn
ipykernel
notebook

## Dataset Description
This dataset used is that of 100 high school students. The dataset includes their student id, age, gpa, gender, major, and year.
